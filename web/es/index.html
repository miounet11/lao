Automatización de navegador para agentes de IA, clientes MCP y terminal

Convierte tu sesión real de Chrome en una API local de navegador.

iatlas-browser evita el problema típico de recrear autenticación en un navegador nuevo. En su lugar, conecta tus herramientas con el navegador que ya estás usando.

Open API

Genera tu API key con email

El servicio público está disponible en https://miaoda.vip/v1/. Cada email recibe una API key con 1000 solicitudes diarias.

Resumen de API

  • Auth: `Authorization: Bearer <apiKey>`
  • Límite: 1000 solicitudes por día
  • Endpoints: `POST /v1/register`, `GET /v1/usage`, `POST /v1/open`

Prueba las dos superficies del API alojado desde esta misma portada

Este laboratorio reúne el subconjunto alojado en un solo lugar. Sirve para obtener URLs públicas y ejecutar adaptadores de solo lectura seguros para remoto. Las tareas que dependen de tus pestañas reales, tu login o el contexto del navegador siguen perteneciendo al runtime local.

Primeros pasos

Instalar, conectar y ejecutar

1. Instalar

curl -fsSL https://miaoda.vip/install.sh | bash

2. Cargar la extensión

chrome://extensions/

Selecciona ~/.iatlas-browser/extension.

3. Iniciar el daemon

iatlas-browser daemon
iatlas-browser doctor

4. Ejecutar comandos

iatlas-browser open https://example.com
iatlas-browser snapshot -i
