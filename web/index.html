miaoda.vip × iatlas-browser

Your browser session, exposed as a local tool layer.

Install from one command, load the Chrome extension, then use iatlas-browser from the terminal, MCP clients, or the local HTTP API.

Download install.sh Download extension zip GitHub

1. One-line install

This installs the repo into ~/.iatlas-browser/src, builds it, and bootstraps local files.

curl -fsSL https://miaoda.vip/install.sh | bash

2. Load the extension

After install, load the unpacked extension in Chrome:

chrome://extensions/

Use the generated path: ~/.iatlas-browser/extension

3. Start local runtime

iatlas-browser daemon
iatlas-browser doctor

4. Use immediately

iatlas-browser open https://example.com
iatlas-browser snapshot -i
iatlas-browser mcp-config cursor
iatlas-browser api-guide

MCP quick start

cursor.json

Paste this into Cursor, Claude Desktop, or another MCP-compatible tool.

{
  "mcpServers": {
    "iatlas-browser": {
      "command": "npx",
      "args": ["-y", "iatlas-browser", "--mcp"]
    }
  }
}

Local API quick start

examples.sh

The daemon exposes a local HTTP API on 127.0.0.1:19824.

curl -s http://127.0.0.1:19824/status

curl -s http://127.0.0.1:19824/command \
  -H "Content-Type: application/json" \
  -d '{"id":"demo-open","action":"open","url":"https://example.com"}'
